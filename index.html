Using your Smartphone, look at an object in the realy world, Identify the object and get Information on it.

Objects can be natural or man-made. Examples: View a tree, plant, fruit, flower, or rock, get the name, type, etc. View a tool or instrument, get the name and how to use it.

FAQ

What if the object does not exist in the Augmented Reality Database?
You can help out by taking a picture of the object, describe it, add tags of what type of object it is, where it was found, etc. Other people can help by viewing and adding comments.
Why would I want to help ID objects that are not in the Database?
Share knowledge. Earn Points. Make collections of similar objects. Be the ID expert on a given category.
